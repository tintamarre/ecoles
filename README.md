# ecoles

> **Note** : Ce projet est une initiative personnelle de Martin Erpicum et n'est pas un projet officiel de la Fédération Wallonie-Bruxelles.

Application web permettant de consulter les établissements d'enseignement de la Fédération Wallonie-Bruxelles.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run serve
```

### Compile and Minify for Production

```sh
npm run build
```
